# SB Credit Card & Gift Card Tracker

A personal finance dashboard to track credit card spending, gift card balances (Amazon/Flipkart), cashback estimates, and savings from offers.

Built with React, Tailwind CSS, and shadcn/ui components. All data stays in your browser via localStorage.

## Quick Start

```bash
npm install
npm run dev
```

Open [http://localhost:5173](http://localhost:5173)

## Features

- Log credit card transactions with date, description, amount, and savings
- Buy and track gift cards (Amazon / Flipkart)
- Spend gift cards and auto-deduct balances
- Dashboard with 7 summary metrics (card utilization, gift card balances, monthly spend, cashback, savings)
- Data persists across sessions via localStorage
